🚀 License Management System (Licensify)

📌 Overview

Licensify is a desktop-based License Management System developed using JavaFX and MySQL that simplifies software license administration through a secure, user-friendly, and centralized platform. The application enables administrators to create, manage, track, and monitor licenses efficiently while maintaining data security and system integrity.

This project was developed to automate manual license management processes, reduce administrative effort, and provide a modern dashboard experience for both administrators and users.
